Умный помощник для сборки ПК
Описание проекта
Мобильное приложение для автоматизированного подбора совместимых комплектующих ПК под задачи и бюджет пользователя. Приложение упрощает процесс сборки ПК для обычных пользователей за счет автоматизации подбора компонентов.

Основные функции
Каталог товаров с 9 категориями комплектующих

ИИ-помощник для консультации по сборке ПК

Корзина с управлением количеством товаров

Оформление заказов с автоматической генерацией чеков

Гостевой режим без обязательной регистрации

Админ-панель для управления товарами и заказами

Продвинутая фильтрация по характеристикам товаров

Технологический стек
Клиент: Kotlin, Android SDK

Бэкенд: Firebase (Firestore, Authentication, Storage)

Архитектура: MVP с использованием Android Components

Установка и запуск
Клонировать репозиторий

Открыть проект в Android Studio

Добавить файл google-services.json в папку app/

Собрать и запустить приложение

Тестовые аккаунты
Администратор: q@gmail.com (любой пароль)

Пользователь: любая регистрация через email или Google

Гость: вход без регистрации через кнопку "Войти как гость"

Особенности реализации
Динамические формы для разных категорий товаров

Автоматическая генерация HTML-чеков

Гибкая система аутентификации (email, Google, гость)

Контекстный интерфейс для разных ролей пользователей
